API GET Request from SBDB

Make an API GET request to NASA’s Small-Body Database query API

We request three fields:

- spkid: unique identifier for the object

- rms: root-mean-square residual of the orbit fit

- last_obs: date of the most recent observation

# Make an API GET request to sbdb_query.api

fields <- "rms,data_arc,spkid,last_obs,n_obs_used"
limit <- 1000 # limit the number of records returned

resp <- request("https://ssd-api.jpl.nasa.gov/sbdb_query.api") |>
  req_url_query(
    fields = fields,
    limit = limit) |>
  # Prevent the request from stopping execution on HTTP errors
  req_error(is_error = function(resp) FALSE) |>
  req_perform()
# Check HTTP status code:
# 200 indicates a successful request
resp$status_code
## [1] 200

Tidy the data

# Parse the JSON response into an R list
data <- resp_body_json(resp)
# Inspect the field names returned by the API
str(data$fields, max.level = 2)
## List of 5
##  $ : chr "rms"
##  $ : chr "data_arc"
##  $ : chr "spkid"
##  $ : chr "last_obs"
##  $ : chr "n_obs_used"
# The API returns the data as a list of rows (one list per object).
# do.call(rbind, ...) binds all rows together into a single matrix-like object.
# This is necessary to convert the nested JSON structure into rectangular data.
df <- as_tibble(
  do.call(rbind, data$data),
  .name_repair = "minimal"
)

# Assign column names using the metadata returned by the API
colnames(df) <- data$fields

# Some columns are stored as lists due to missing values in the API response.
# unlist() flattens each column into a standard atomic vector,
# which is required before converting data types.

df <- df |>
  mutate(
    spkid = unlist(spkid),
    rms = unlist(rms),
    last_obs = unlist(last_obs),
    data_arc = unlist(data_arc),
    n_obs_used = unlist(n_obs_used)
  )

More data tidying

# Convert columns to appropriate data types:
#  - spkid remains character
#  - rms is numeric
#  - n_obs_used is numeric
#  - data_arc is numeric
#  - last_obs is converted to Date
# Missing Data: 
# Empty strings in last_obs are replaced with NA before conversion
# compare to an ifna() in Excel
df <- df |>
  mutate(
    spkid = as.character(spkid),
    rms = as.numeric(rms),
    n_obs_used = as.numeric(n_obs_used),
    data_arc = as.numeric(data_arc),
    last_obs = na_if(last_obs, ""),
    last_obs = as.Date(last_obs, format = "%Y-%m-%d")
  )

Plot Distribution of Orbit RMS

The RMS of the orbit of a small body measures how well the computed orbit fits the observational data. Lower RMS values indicate a better orbit fit.

ggplot(df, aes(x = rms)) +
  geom_histogram(binwidth = 0.02, fill = "blue", color = "black") +
  labs(title = "Distribution of Orbit RMS values of Small Bodies",
       x = "RMS",
       y = "Count") +
  theme_minimal()

Observation arc length reflects the total time span between the first and most recent observations, not the age of the most recent measurement. Objects with arcs exceeding 50,000 days were typically discovered in the 19th century and have been re-observed repeatedly with modern instrumentation, resulting in highly constrained orbital solutions and very low rms values.

rms_cutoff <- quantile(df$rms, 0.05, na.rm = TRUE)

ggplot(df, aes(x = data_arc, y = rms)) +

    data = subset(df, rms <= rms_cutoff),
